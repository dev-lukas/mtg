Challange Regeln

Jeder Teilnehmer startet die Challange mit einem Precon-Commander Deck. Wenn nun EDH Runden gespielt werden mit ausschließlich Challange Decks, werden drei zufällige Challanges ausgelost. Nun wird normal die Partie gespielt. Punkte werden danach wie folgt erhalten:
  • Für das Teilnehmen an der Runde erhält jeder Spieler 3 Punkte
  • Der Spieler der eine Challange erfüllt erhält die spezifizierten Challange-Punkte
  • Spieler die nicht teilgenommen haben erhalten 2 Punkte
  • Per Mehrheitsentscheid darf ein weiterer Punkt verteilt oder abgezogen werden
Punkte können wie folgt zum upgraden eines Decks verwendet werden:
  • Ganze Decks können immer zu anderen Precons getauscht werden. Hierbei verfallen alle bisherigen Punkte
  • Karten dürfen ohne Punkteausgabe durch Basic Lands ausgetauscht werden
  • Karten können geupgraded werden zum Punkte-Preis des Preis Checkers
